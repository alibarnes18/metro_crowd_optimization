# Contributing to MCOS

Thank you for your interest in helping with the Metro Crowd Optimization System!

## How to Contribute

1. **Fork the Repository**: Create your own copy of the project.
2. **Setup Locally**: Follow the instructions in the README to get the project running.
3. **Commit Your Changes**: Use descriptive commit messages.
4. **Push and Pull Request**: Submit your changes for review.

## Code Standards

- **Frontend**: Follow React best practices. Use Tailwind CSS for styling.
- **Backend**: Maintain the modular structure in the `processing` and `engine` folders.
- **Language**: Use Azerbaijani for UI-facing elements and English for internal code/documentation.

## Reporting Issues

Use GitHub issues to report bugs or suggest enhancements. Please provide as much detail as possible.

---

We appreciate every contribution, from fixing typos to implementing complex simulation algorithms!
